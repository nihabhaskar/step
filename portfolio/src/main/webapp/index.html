    Almounds

    Application for the Central California Almond Growers Association

    Almounds is an application designed for the Central California Almond Growers Association (CCAGA) in order to make the lives of workers and administrators exponentially easier and more organized by streamlining tasks that are normally done by hand. The application offers a dynamic grid that allows for updates pending the filling and emptying of stockpiles, utilizing an algorithm that optimizes resources when a dropoff location is obtained for a truckload, and providing an all-in-one organized user interface for the workers and administrators at CCAGA.



    Investigation of AFRINIC IPv4 Address Theft

    Early Research Scholars Program

    During ERSP, we conducted an experiment that analyzed shifts of IPv4 addresses away from Africa to identify organizations that may be misappropriating unused AFRINIC IPv4 addresses. We were able to conduct multiple experiments to explore the properties that make an IP address considered suspicious by gathering and comparing data from 3 datasets: Delegation files, BGP Data, and WHOIS data. Using the information from each dataset, we were able to first create scripts that analyzed address blocks that should be in Africa and then determined, through the BGP data announcements who the actual owner of the address block was and if verify if it was truly in use in Africa.
